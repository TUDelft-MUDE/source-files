# Source files

Repository where binary source files are stored using LFS for use in other MUDE repositories.

All large (binary) files should be added in the `file/` directory in which they are tracked using Git LFS.

Include files in other documents with the link format:

```
https://github.com/TUDelft-MUDE/source-files/raw/main/file/<filename>
```

Further instructions available here: https://mude.citg.tudelft.nl/teacher/LFS.html
